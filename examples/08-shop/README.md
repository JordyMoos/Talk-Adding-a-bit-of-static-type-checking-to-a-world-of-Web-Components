# Elm Polymer Shop

The polymer shop but then partially in Elm!

# Run this

Clone this
```
git clone https://github.com/JordyMoos/elm-polymer-shop.git
```
```
cd elm-polymer-shop
```

Install dependencies
```
bower install
yarn install
```

Run the app
```
yarn dev:build
yarn dev
```

Goto `[::1]:8888` in your browser.
